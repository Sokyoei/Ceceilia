# C/C++ 内存分区

```
+-------+---------------------------------------------------+ high address
| os    | system kernel                                     |
+-------+---------------------------------------------------+
| env   | env variable argv                                 |
+-------+---------------------------------------------------+
| stack | compile/system args local variable                |
+-------+---------------------------------------------------+
|   ↓   |                                                   |
|       | unuse                                             |
|   ↑   |                                                   |
+-------+---------------------------------------------------+
| heap  | malloc/calloc/realloc/free new/delete(free store) |
+-------+---------------------------------------------------+
| bss   | uninit global variable                            |
+-------+---------------------------------------------------+
| data  | init global variable  string/other constant       |
+-------+---------------------------------------------------+
| text  | code                                              |
+-------+---------------------------------------------------+ low address
```
